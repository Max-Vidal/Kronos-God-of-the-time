# Kronos: God of the time
Kronos: God of the time es un juego desarrollado en Java utilizando la biblioteca Swing para la interfaz gráfica de usuario.

## Características
- Interfaz gráfica desarrollada con Java Swing.
- Lógica de juego implementada en Java puro.
- Estructura de proyecto modular y organizada.

## Requisitos del sistema
- Java Development Kit (JDK) 8 o superior instalado.
- Sistema operativo compatible con Java (Windows, macOS, Linux).
